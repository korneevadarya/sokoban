# sokoban

Классическая игра-головоломка "Сокобан", реализованная на Python с использованием библиотеки Pygame.

## Описание игры

Сокобан - это логическая игра, в которой игроку нужно расставить все ящики на специальные места (цели). Игрок может толкать ящики, но не может их тянуть. Цель игры - разместить все ящики на целевых клетках.

## Управление

- **Стрелки (← ↑ → ↓)** - перемещение игрока
- **Backspace** - отмена последнего хода
- **Space** - возврат отмененного хода
- **R** - перезапуск текущего уровня

## Особенности

- Автоматический переход на следующий уровень после завершения текущего
- Возможность отмены и возврата ходов (история ходов сохраняется)
- Подсчет количества сделанных ходов и толканий ящиков
- Отображение текущего уровня и прогресса в заголовке окна

## Установка и запуск

### Требования
- Python 3.6 или выше
- Pygame

### Установка зависимостей
```bash
pip install pygame
```
## Структура уровней
Уровни загружаются из файла levels.txt. Формат файла:

- ширина высота
- строка_уровня
- позиция_игрока

## Обозначения в уровне:
Символ	Описание
`#`	    Стена
`-`	    Пустое место
`.`	    Цель (место для ящика)
`$`	    Ящик
`*`	    Ящик на цели
`@`	    Игрок
`+`	    Игрок на цели

## Файлы игры
sokoban.py - основной файл игры

levels.txt - файл с уровнями (создайте его для добавления своих уровней)

borgar.png - файл с графикой (спрайты)

## Создание своих уровней
Откройте файл levels.txt

Добавьте уровень в формате:

- ширина высота
- строка_уровня_длиной_ширина_х_высота
- позиция_игрока

Используйте обозначения из таблицы выше

Можно добавить несколько уровней подряд

## Примечания
Убедитесь, что файл borgar.png находится в той же папке, что и игра

Если файл уровней не найден, игра создаст уровень по умолчанию

При загрузке уровней автоматически проверяется соответствие размера строки уровня заданным размерам

Если в файле уровней есть ошибки, игра использует уровень по умолчанию
